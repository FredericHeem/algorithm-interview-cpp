A collection of *generic* algorithms in C++ 11, useful for job interviews.

## Sorting

* Quick Sort
* Bubble Sort
* Insertion Sort

## Data structures

* Linked List
* Binary Tree

### Linked List

* creation
* push back element
* size
* clear
* reverse
* delete by value

### Binary Tree 

 * create 
 * insert node
 * remove node
 * clear all nodes
 * Breadth first search 
 * Depth first search
 
## How to build and run 

Build and Run:

    $ make
    $ ./QuickSort
    $ ./BubbleSort
